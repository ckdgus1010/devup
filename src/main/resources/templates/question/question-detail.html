📋 Java의 정적(static) 키워드란?

📁 Java 🎯 중 🕒 2025-05-09

📝 문제

정적 키워드(static)는 클래스 수준에서 공유되는 변수를 선언할 때 사용됩니다. 객체가 여러 개여도 static 변수는 하나만 존재합니다.

정답:

static 키워드는 클래스 단위로 메모리를 공유하기 위한 키워드입니다.
static 변수는 모든 인스턴스가 공유하며, static 메서드는 객체 없이 클래스에서 직접 호출할 수 있습니다.
